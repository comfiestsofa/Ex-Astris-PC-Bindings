# 엑스 아스트리스 PC 키 설정
[[English](README.md)] [[日本語](README-日本語.md)] [[**한국어**](README-한국어.md)] [[繁體中文](README-繁體中文.md)] [[简体中文](README-简体中文.md)]

Hypergryph·Gryphline의 3D RPG 게임 [엑스 아스트리스를](https://exa.gryphline.com/ko-kr) 위한 완전 번역된 PC 키 설정입니다.

PlayStation 컨트롤러, XInput 컨트롤러 (DS4Windows 등 포함), 마우스와 키보드에 대응합니다.

Hypergryph·Gryphline이 공식적으로 공개한 간체 중국어 MuMuPlayer 키 설정을 기반으로 제작되었습니다.

(⚠ 저는 해외에서 태어난 한국인이라 한국어가 모국어 수준이 아닙니다. 불자연하거나 이상한 표현이 있으면 죄송합니다.)

# 설치 방법
* "Code" -> "Download ZIP"을 클릭하여 이 저장소를 다운로드하거나, Git 저장소를 클론하세요.
* `install.bat` 또는 `install.sh`를 실행합니다. (⚠ 주의: 기존의 사용자 지정 키 설정은 덮어써집니다)
* 또는 각 키 설정 JSON 파일을 MuMuPlayer에 수동으로 가져올 수도 있습니다.

# 개선 사항
* 한국어로 완전 번역 (영어, 일본어, 번체 중국어, 간체 중국어 포함)
* DS4Windows를 사용해 XInput 컨트롤러를 에뮬레이트할 필요 없이, PlayStation 컨트롤러 네이티브 지원 추가
* 마우스와 키보드 레이아웃을 기반으로 한 통합 컨트롤러 설정 추가. 파티 인원 수에 따라 전환할 필요가 없습니다
* 아날로그 이동 감도 수정
* 2인용 컨트롤러 설정에서 오른쪽 스틱 카메라 조작이 누락되던 문제 수정
* 전투 시 적 타겟 선택용 버튼 추가
* 버튼 라벨 개선
* 모든 버튼 위치 통일
* 이전에 마우스와 키보드 전용이었던 기능을 컨트롤러 설정에 모두 추가
	* 돌아가기/임무
	* 전투 정보
	* 메뉴
	* 엔트로피스 1
	* 엔트로피스 2
	* 엔트로피스 3
	* 상호작용 2
	* 추가 공격
* 컨트롤러·마우스와 키보드 설정 모두에 추가 기능 구현
	* 아이템 창 ("NOUS WAVE" 버튼)
	* 정보 1 (설정 자료 팝업)
	* 정보 2 (적 정보 팝업)
	* 적 1
	* 적 2
	* 적 3
* 마우스·키보드에서는 추가 공격가 모든 캐릭터 슬롯에서 사용 가능하며, 3번째 캐릭터에만 제한되지 않습니다. 옌을 어디에나 배치할 수 있습니다
* 특정 키보드 버튼 (특히 돌아가기/임무, 상호작용 1, 상호작용 2) 을 눌러도 마우스가 자동으로 잠기지 않습니다
