# 『エクスアストリス』PCキー設定
[[English](README.md)] [[**日本語**](README-日本語.md)] [[한국어](README-한국어.md)] [[繁體中文](README-繁體中文.md)] [[简体中文](README-简体中文.md)]

Hypergryph・Gryphlineの3D RPGゲーム[『エクスアストリス』](https://exa.gryphline.com/ja-jp)向けの完全翻訳済みのPCキー設定です。

PlayStationコントローラー、XInputコントローラー（DS4Windowsなど含む）、およびマウスとキーボードに対応しています。

Hypergryph・Gryphlineが公式に公開した簡体字中国語のMuMuPlayerキー設定を基にしています。

(⚠ 日本語は母国語ではないので、不自然や変な表現があってご容赦ください。)

# スクリーンショット
[スクリーンショットはこちらから閲覧できます。](https://drive.google.com/drive/folders/1k7yrsdX55T33MfIIMfmdA6cJZatq4f_s)

# インストール方法
* 「Code」 -> 「Download ZIP」からこのリポジトリをダウンロードするか、Gitリポジトリをcloneしてください。
* `install.bat`または`install.sh`を実行します。（⚠ 注意：既存のカスタムキー設定は上書きされます）
* もしくは、各キー設定のJSONファイルをMuMuPlayerに手動でインポートすることも可能です。

# 改善点
* 日本語に完全翻訳（英語、韓国語、繁体中国語、簡体中国語にも）
* DS4Windowsを使ってXInputコントローラーをエミュレートする必要なく、PlayStationコントローラーのネイティブ対応を追加
* マウスとキーボードのレイアウトを基にした「統合」コントローラー設定を追加。パーティメンバー数に応じて切り替える必要がなくなります
* アナログ移動の感度を修正
* 2キャラのコントローラー設定で右スティックによるカメラ操作が欠落していた問題を修正
* 戦闘時の敵ターゲット選択用ボタンを追加
* ボタンラベルを改善
* 全ボタンの配置を統一
* 以前マウスとキーボード専用だった機能をコントローラー設定にすべて追加
	* 戻る/任務
	* 戦闘情報
	* メニュー
	* エントロピース1
	* エントロピース2
	* エントロピース3
	* 調べる2
	* 追加攻撃
* コントローラー・マウスとキーボード設定両方の追加機能
	* アイテム欄（「NOUS WAVE」ボタン）
	* 情報1（設定資料ポップアップ）
	* 情報2（敵情報ポップアップ）
	* 敵1
	* 敵2
	* 敵3
* マウスとキーボードでは、「追加攻撃」が全キャラクタースロットで使用可能になり、3番目のキャラクターに限定されなくなりました。雁をどこにでも配置可能
* 特定のキーボードボタン（特に戻る/任務、調べる1、調べる2）を押しても、マウスが自動でロックされなくなりました
